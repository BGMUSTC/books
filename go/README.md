# Go 语言

Go 是一个完全开源的编程语言，用于构建简单、可靠且高效的软件。


* **官方地址**为 [golang.google.cn](https://golang.google.cn) 或 [golang.org](https://golang.org) （后者需要科学上网进行访问）。
